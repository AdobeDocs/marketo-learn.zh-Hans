# 对话式登陆页面

只需单击几下，即可将Dynamic Chat会话流安装到任何登陆页面(Marketo登陆页面或自定义)中，并转换任何链接或按钮以触发会话参与。

您可以解锁的用例 — 

- 允许访客预订会议
- 提出资格问题
- 根据访客选择/兴趣推荐的内容
- 共享文档并跟踪参与
